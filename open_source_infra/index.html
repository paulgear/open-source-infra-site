Open Source Infrastructure List

I've done many talks about working on an open source infrastructure for an open source project. The following is a list of open source infrastructures that I'm familiar with:

  • OpenStack: http://docs.openstack.org/infra/system-config/ (the one I work on)
  • Debian: https://dsa.debian.org/
  • Mozilla: https://wiki.mozilla.org/ReleaseEngineering/PuppetAgain
  • Fedora: https://fedoraproject.org/wiki/Infrastructure
  • Jenkins: http://jenkins-ci.org/content/come-join-infra-team
  • Ubuntu: https://code.launchpad.net/~canonical-losas (Staging and development) and https://jujucharms.com/store (Curated production) and https://mojo.canonical.com/ (Continuous Delivery)
